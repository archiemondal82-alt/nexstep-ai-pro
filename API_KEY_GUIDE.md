### Problem: "Invalid API Key"

**Solutions:**
1. **Check for spaces**
   - Make sure no spaces before/after the key
   - Should look like: `AIzaSy...` (no spaces)

2. **Copy the entire key**
   - Keys are long (40+ characters)
   - Make sure you copied everything

3. **Try creating a new key**
   - Delete old key
   - Create a new one
   - Use the new one

### Problem: "API Key Not Working"

**Solutions:**
1. **Wait 1 minute**
   - New keys take ~30 seconds to activate
   - Refresh the page

2. **Check your Google account**
   - Make sure you're signed in
   - Try signing out and back in

3. **Create in "new project"**
   - Use "Create API key in new project" option
   - Not "existing project"

### Problem: "Cannot Access Google AI Studio"

**Solutions:**
1. **Check internet connection**
2. **Try different browser**
   - Chrome (recommended)
   - Firefox
   - Edge
3. **Disable VPN** (if using one)
4. **Clear browser cache**
